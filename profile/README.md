## Getting Started

1. Obtain **EarTrumpet** from the official repository.  
2. Launch **EarTrumpet** directly from the extracted folder (no installation required).  
3. Right-click the system tray icon to access **EarTrumpet** main panel.  
4. Adjust per-app volumes, change output devices, or set default communication devices.  
5. Configure **EarTrumpet** hotkeys and theme preferences.  
6. (Optional) Carry **EarTrumpet** on a USB drive for portable audio management.
